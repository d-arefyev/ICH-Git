Создание простых маршрутов
1. Создайте новый проект Node.js и установите Express.js, если вы еще этого не сделали.
2. Создайте файл 'index.js' и откройте его в текстовом редакторе.
3. Импортируйте Express и создайте экземпляр приложения.
4. Создайте следующие маршруты:
• Корневой маршрут (/'), который возвращает строку "Welcome to my site!".
• Маршрут для получения списка продуктов (/products*), который возвращает строку "List of products".
5. Откройте браузер и перейдите по адресу http://localhost:3000 для проверки корневого маршрута и
http://localhost:3000/products для проверки маршрута продуктов.
